# @community-gaming/ws-gateway

WebSocket server for realtime features: presence, matchmaking, chat.

## Features

- **Presence**: Online/offline status, typing indicators
- **Rooms**: Join/leave channels and lobbies
- **Matchmaking**: Real-time queue updates
- **Chat**: Low-latency message delivery

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
