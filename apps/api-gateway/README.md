# @community-gaming/api-gateway

Fastify-based API Gateway with authentication, rate limiting, and service routing.

## Features

- **Authentication**: JWT-based auth with OIDC support
- **Authorization**: RBAC/ABAC
- **Rate Limiting**: Token bucket algorithm
- **Validation**: Zod schema validation
- **Observability**: OpenTelemetry integration

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
