# @community-gaming/config

Shared configuration for TypeScript, ESLint, and build tools.

## Includes

- `tsconfig.base.json`: Base TypeScript config
- `tsconfig.nextjs.json`: Next.js specific config
- `tsconfig.react.json`: React library config
- `eslint-base.js`: Base ESLint config
- `eslint-nextjs.js`: Next.js ESLint config
